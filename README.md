Canny Edge Detection is a multi-stage algorithm developed by John F. Canny in 1986. It aims to detect a wide range of edges in images by:

1. **Noise Reduction**: Applying Gaussian blur to smooth the image.
2. **Gradient Calculation**: Finding intensity gradients using Sobel filters.
3. **Non-Maximum Suppression**: Thinning edges to retain only the sharpest ones.
